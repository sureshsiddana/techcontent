# Azure App Services

Azure App Services is a fully managed platform for building, deploying, and scaling web apps and APIs.

## Key Concepts
- Web Apps, API Apps, and Mobile Apps
- Deployment Slots
- Scaling and Load Balancing
- Custom Domains and SSL
- Continuous Deployment

## Real-world Project Example
Deploying a production-ready web application with staging and production slots, custom domain, and SSL.

## Interview Questions & Answers
**Q1: What is Azure App Service?**
A: A platform-as-a-service (PaaS) for hosting web apps, REST APIs, and mobile backends.

**Q2: How do deployment slots work?**
A: They allow you to host multiple versions (e.g., staging, production) and swap with zero downtime.

**Q3: How do you scale an app in App Services?**
A: Use manual or auto-scaling based on demand.
