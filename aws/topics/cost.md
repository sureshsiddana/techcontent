# AWS Cost Management

## Key Concepts
- **Cost Explorer:** Visualize, understand, and manage AWS costs and usage.
- **Budgets:** Set custom cost and usage budgets and receive alerts.
- **Billing Dashboard:** Centralized view of AWS spending.
- **Savings Plans & Reserved Instances:** Reduce costs with long-term commitments.
- **Trusted Advisor:** Cost optimization recommendations.

## Real-World Example
- **Budget Alerts:** Set up alerts to notify when spending exceeds thresholds.
- **Cost Allocation Tags:** Track costs by project, department, or environment.

## Interview Q&A
**Q: What is AWS Cost Explorer?**
A: Cost Explorer helps visualize and analyze AWS spending patterns.

**Q: How can you optimize AWS costs?**
A: Use Savings Plans, Reserved Instances, right-size resources, and monitor with Cost Explorer and Budgets.

**Q: What are cost allocation tags?**
A: Tags that help organize and track AWS costs by resource, project, or team.
