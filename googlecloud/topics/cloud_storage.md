# GCP Cloud Storage

## Key Concepts
- **Object Storage:** Store and retrieve any amount of data at any time.
- **Buckets:** Containers for objects/files.
- **Storage Classes:** Standard, Nearline, Coldline, Archive for cost optimization.
- **Lifecycle Management:** Automate data retention and deletion.
- **Access Control:** IAM, ACLs, signed URLs.

## Real-World Example
- Hosting static website assets and backups in Cloud Storage buckets.

## Interview Q&A
**Q: What is Cloud Storage?**
A: GCP's object storage service for unstructured data.

**Q: How do you secure data in Cloud Storage?**
A: Use IAM, ACLs, and signed URLs for access control.
